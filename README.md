# Whisper

## Requirements

- Python 3.13.x

## create venv

### MacOS

```zsh
python3 -m venv .venv
source .venv/bin/activate
```

### Windows

```bash
python -m venv .venv
.venv\Scripts\activate.bat
```

## install dependencies

```zsh
pip install -r requirements.txt
```

## run

```zsh
python src/main.py
```
